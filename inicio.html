Información General

Objetivos OVI


*El presente OVI se diseña para dar al estudiante una primera introducción a la interacción con el lenguaje html5 *Conocer las diferentes etiquetas del lenguaje y su respectiva implementación
*Insertar diferentes estilos y fuentes para ser utilizadas en la página web
*Crear enlaces a las diferentes páginas del sitio
*Diseñar diferentes estilos a cada uno de los elementos de la página

Objetivos Curso


*Comprender el funcionamiento del lenguaje html5 junto con css y javascript
*Aprender el concepto de cada una de las librerías y su inserción al inicio del documento
*Diseño y creación de un sitio web que se pueda visualizar en los diferentes navegadores
*Conocer y manejar el editor de código sublime text

Temáticas Curso


*Introducción a HTML5
*Estructura Básica de HTML5
*Aplicación de elementos de HTML5 a un proyecto de diseño web
*Introducción a CSS3
*Estructura básica de CSS3
*Aplicación de elementos de CCS3 a un proyecto de diseño web
